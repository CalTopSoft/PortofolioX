PORTAFOLIO DE CÁLCULO INTEGRAL
================================

ESTRUCTURA DEL PROYECTO:
- index.html              Página principal con portada
- styles.css              Estilos principales
- pages/                  Páginas de contenido
  - parcial1.html         Primer parcial (7 temas)
  - parcial2.html         Segundo parcial (5 temas)
  - informacion.html      Tu información personal
- assets/                 Recursos multimedia
  - images/               Imágenes de gráficas
  - icons/                Iconos SVG
  - icoz.png              Favicon

INSTRUCCIONES:
1. Crea la estructura de carpetas exactamente como se muestra arriba
2. Copia cada archivo HTML en su ubicación correspondiente
3. Guarda los iconos SVG en assets/icons/
4. Si tienes imágenes, guárdalas en assets/images/
5. Abre index.html en tu navegador para ver el resultado
